# Translation Bridge 🏎️

أداة بسيطة وخفيفة للويندوز تترجم كلامك من عربي إلى إنجليزي فوري وتلصقه بالديسكورد، أو داخل اللعبة، أو أي شات تستخدمه. 

الفكرة من هذا البرنامج إنه يحل مشكلة "الترجمة الحرفية والآلية" اللي تفشلنا بالغالب. البرنامج مربوط بنموذج **Claude 3.5 Haiku** (عن طريق OpenRouter) ومبرمج إنه يفهم الـ Context (السياق) ويفهم كلامنا العادي العامي والسعودي، وحتى مصطلحات القيمنق (مثل: يب، ماش، بيض، ندرعم) ويحولها لإنجليزي حقيقي كأنك نيتڤ!

## ✨ الميزات

- **ترجمة سياقية تفهم لغتنا**: تترجم المعنى والمقصد، مو ترجمة قوقل الحرفية.
- **اختصار كيبورد (Ctrl+Shift+T)**: مو لازم تفتح البرنامج كل شوي! وأنت تلعب أو تسولف، اضغط الاختصار، بيطلع لك مربع صغير فوق الماوس مباشرة (يدعم الشاشات المتعددة)، اكتب، اضغط إنتر، وهو بيترجم ويلصق الكلام في الشات ويرسله عنك!
- **٣ أوضاع للتشغيل**:
  1. نسخ للـ Clipboard بس 📋
  2. نسخ ولصق في الشات 📋→📌
  3. نسخ ولصق وإرسال (يضغط إنتر بدالك) 📋→📌→🚀
- **شغال بالخلفية**: تقدر تقفل النافذة الأساسية وراح يتخبى تحت بشريط المهام (System Tray) ويبقى الاختصار شغال معاك.
- **يرجعك للنافذة صح**: مجرد ما يخلص ترجمة، يرجع الفوكس (Focus) للعبتك أو الشات عشان يلصق النص بالضبط مكان ما كنت.

## 🚀 طريقة التثبيت والتشغيل

### لو تبي تشغله كسكريبت بايثون (للمطورين)
تأكد إن عندك بايثون (Python 3.10+) مثبت بجهازك.

1. حمل الملفات وافتح التيرمنال في المجلد.
2. ثبت المكتبات المطلوبة:
   ```cmd
   pip install -r requirements.txt
   ```
3. شغل البرنامج:
   ```cmd
   python chat_bridge.py
   ```

### كيف أجهزه لنسخة EXE؟
في حال ودك تخليه تطبيق ينفتح بضغطة زر كبرنامج مستقل (عشان تشاركه مع أصحابك أو تحطه بسطح المكتب):
بس اضغط على ملف `build.bat` مرتين أو شغله بالتيرمنال:
```cmd
.\build.bat
```
بعد ما يخلص، بتحصل البرنامج كامل في مجلد `dist/` جاهز للاستخدام!

## ⚙️ وش احتاج عشان يشتغل؟
أول ما تشغل البرنامج، بيطلب منك مفتاح **API Key**. 
- رح لموقع [OpenRouter.ai](https://openrouter.ai/keys) وسجل دخولك، وخذ لك API Key.
- الصقه في البرنامج وهو بيحفظه محلياً في جهازك (ما راح ينرفع لأي مكان ولا أحد يقدر يشوفه).
- تقدر تعدل الاختصار `Ctrl+Shift+T` لأي اختصار تبيه من علامة الإعدادات ⚙️ جوة البرنامج.

## 🔒 الخصوصية
المفاتيح والإعدادات الخاصة فيك تنحفظ بملفات مخفية (`.api_key` و `.config.json`) محلياً بجهازك فقط. إذا كنت بترفع تعديلاتك على قيت هب، تأكد إنها ما تنرفع (وهي مضافة أساساً في الـ `.gitignore`).
