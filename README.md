# Translation Bridge

أداة لنظام ويندوز تترجم النصوص من العربية إلى الإنجليزية ترجمة سياقية باستخدام نموذج Grok 4.1 Fast (عبر OpenRouter)، ثم تلصق الناتج في نافذة المحادثة أو اللعبة تلقائيا.

تعالج هذه الأداة مشكلة الترجمة الحرفية، إذ تفهم السياق والعبارات العامية ومصطلحات الألعاب وتحولها إلى إنجليزية طبيعية.

## المزايا

- ترجمة سياقية تراعي المعنى والمقصد بدلا من الترجمة الحرفية.
- اختصار لوحة المفاتيح (Ctrl+Shift+T) لفتح نافذة ترجمة سريعة فوق مؤشر الفأرة مباشرة، مع دعم الشاشات المتعددة.
- ثلاثة أوضاع تشغيل:
  1. النسخ إلى الحافظة فقط.
  2. النسخ واللصق في المحادثة.
  3. النسخ واللصق والإرسال تلقائيا.
- العمل في الخلفية عبر شريط النظام (System Tray) مع بقاء الاختصار فعالا.
- إعادة التركيز تلقائيا إلى النافذة السابقة بعد الترجمة.

## التثبيت والتشغيل

### التشغيل كسكربت بايثون

المتطلبات: Python 3.10 أو أحدث.

```cmd
pip install -r requirements.txt
python chat_bridge.py
```

### بناء ملف تنفيذي (EXE)

```cmd
.\build.bat
```

يُنشأ الملف التنفيذي في مجلد `dist/`.

## طريقة الاستخدام

1. اضغط بالفأرة في حقل الكتابة داخل المحادثة أو اللعبة.
2. اضغط الاختصار `Ctrl + Shift + T` (قابل للتعديل من الإعدادات).
3. اكتب النص العربي في النافذة المنبثقة ثم اضغط Enter.
4. يُترجم النص ويُلصق ويُرسل تلقائيا حسب الوضع المحدد.

## الأمان

مفتاح API يُحفظ محليا في ملف مخفي على الجهاز فقط، ولا يُرسل إلى أي خادم خارجي.
